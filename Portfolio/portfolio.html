•Web Journal Application

This web application, meticulously built from scratch, combines front-end and back-end development, adeptly handling multiple website requests and responses. It operates efficiently on a local host, showcasing a blend of essential functionalities and user-friendly features. Among its key offerings are Journal Creation, where users can generate entries with personalized titles and content, and Automatic Date Title, which provides a date-based title for entries lacking user-specified titles. Additionally, the application facilitates Journal Retrieval, allowing users to access their past entries with ease. It also includes features for Journal Editing, enabling modifications to existing entries, and Journal Deletion, providing users with the option to remove entries as required.

•Simon's Sequence Web Challenge

"Simon's Sequence Web Challenge" is a memory-testing web game based on the classic Simon game, primarily designed for desktop or laptop users. Using JavaScript and jQuery, this project showcases dynamic patterns and error detection, engaging players in a test of memory and attention. Experience the power of JavaScript Math functions and higher-order functions in a game that challenges you to match increasingly complex sequences. Perfect for quick and fun mental exercise!
